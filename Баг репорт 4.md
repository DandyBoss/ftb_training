Название: Утечка персональных данных пассажиров — все роли видят полную информацию
Шаги воспроизведения:
1. Залогиниться в аккаунт с любой ролью
2. Попасть на любую страницу рейсов https://ftb.mentorpiece.org/flights?pageNo=0
3. Нажать на Passengers в рейсе с уже забронированными местами

Отображается информация только об ID, First Name, Last Name
Фактический результат:
Доступна вся информация об пассажирах для всех ролей
