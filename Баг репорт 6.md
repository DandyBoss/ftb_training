Название: Возможность выбора прошедшей даты для вылета рейса
Шаги воспроизведения:
1. Залогиниться в аккаунт с любой ролью
2. Перейти в раздел Search https://ftb.mentorpiece.org/flight/search
3. В поле "Дата вылета" выбрать прошедшую дату
4. Заполнить остальные поля формы
5. Нажать кнопку поиска
Ожидаемый результат:
Невозможность выбора прошедших дат для времени вылета
Фактический результат:
Имеется возможность выбрать прошедшую дату для времени вылета
